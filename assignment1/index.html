1. What is HTML and what does it stand for?
HTML stands for Hypertext Markup Language. It is the standard markup language used to create and design web pages and applications. HTML elements are the building blocks of web pages, and by combining HTML with CSS (Cascading Style Sheets) and JavaScript, it's possible to create visually engaging and interactive websites.


2. What is the difference between HTML and XHTML?
HTML (Hypertext Markup Language) and XHTML (Extensible Hypertext Markup Language) both serve the purpose of structuring content for the web. The main differences are: 
    - HTML is based on SGML (Standard Generalized Markup Language) and doesn't require strict syntax rules. 
    - XHTML is an application of XML (Extensible Markup Language) and thus requires stricter syntax rules such as closing all tags, writing all tags and attributes in lowercase, and quoting all attribute values.
3. What are the basic building blocks of an HTML document?
The basic building blocks of an HTML document are:
    HTML Tags: These are the elements wrapped in angle brackets like < html >, < body >, < div >, and more.
    Attributes: They provide additional information about HTML elements, such as class, id, style, etc.
    Content: This is the text or media placed between the tags that appear on the webpage.

19. What is the difference between block-level and inline elements in HTML?
Block-level elements: These elements take up the full width available and start on a new line. Examples include < div >, < p >, < h1 >, < section >, and < article >.
Inline elements: These elements take up only as much width as their content requires and do not start on a new line. Examples include < span >, < a >, < img >, and < strong >.
